# **App Name**: SpeechAce

## Core Features:

- Article Listing: Display a list of articles with filtering by categories such as 'Speech Therapy,' 'Exercises,' and 'Parenting Tips'.
- Article Detail: Display the full content of an article, including text, images, and metadata.
- Dynamic Routing: Automatically generate new pages from a local JSON file or `/content` folder. Dynamically create titles and descriptions for each page
- SEO Optimization: Include JSON-LD metadata in each article for better search engine visibility. configure image optimization or lazy loading.

## Style Guidelines:

- Primary color: Soft mint green (#E0F2F1) for a calming and approachable feel.
- Accent color: Warm coral (#FFB74D) for call-to-action buttons and interactive elements.
- Text color: Deep gray (#37474F) for enhanced readability.
- Body and headline font: System sans-serif fonts to minimize loading times and ensure cross-device compatibility.
- Clean and airy layout with ample white space, rounded corners (12-16px), and large, mobile-friendly buttons.
- Prioritize a mobile-first design approach to ensure optimal viewing experience on smartphones.
- Simple, clear icons to visually represent categories and actions, enhancing user experience.
